# Core Beliefs

1. Users pay for clarity, not for mystical vagueness.
2. Traditional visual cues can coexist with a modern, data-forward product voice.
3. The preview must build trust before asking for payment.
4. Sensitive birth data should feel protected at every step.
5. Mobile flow quality matters more than desktop polish in the MVP.
6. Category structure makes fortune content easier to browse and sell.
7. The calculation layer must remain deterministic even if the narrative layer evolves.
